# 한국사 전사 배틀 RPG · 바이브코딩 교재

한국사 퀴즈 RPG 게임을 수업용 교재 화면과 함께 볼 수 있도록 정리한 정적 웹 프로젝트입니다.

## 파일 구성

- `index.html`: 교재형 첫 화면
- `game.html`: 한국사 전사 배틀 RPG 실행 화면

## GitHub Pages로 공개하기

1. GitHub에 새 저장소를 만듭니다.
2. 이 폴더의 파일을 저장소에 올립니다.
3. 저장소의 `Settings` → `Pages`로 들어갑니다.
4. `Source`를 `GitHub Actions`로 선택합니다.
5. 생성된 `https://아이디.github.io/저장소이름/` 주소로 접속합니다.

## 수업 활용 아이디어

- 게임 제목과 캐릭터 스킨 이름 바꾸기
- 한국사 퀴즈 문제 추가하기
- 오답 해설을 더 쉽게 고치기
- 모바일 화면에서 버튼 크기 조정하기
