한국사 RPG를 먼저 플레이해요

목표: 퀴즈를 맞혀 공격권을 얻고, 적과 보스를 물리치며 한국사 개념을 복습합니다.

게임은 원본 HTML을 그대로 살려 별도 페이지로 넣었습니다. 수업에서는 먼저 한 번 플레이한 뒤, “어떤 기능을 더 넣고 싶은지”를 말로 적어보면 좋습니다.

게임 시작하기 수업 미션 보기
1단계

Codex에게 부탁할 말을 정해요

  1. 게임 제목, 색깔, 캐릭터 이름처럼 눈에 보이는 부분부터 바꿔 봅니다.
  2. 퀴즈 문제를 더 추가하거나, 틀렸을 때 나오는 설명을 친절하게 바꿉니다.
  3. 친구가 휴대폰으로 열어도 잘 보이는지 확인합니다.
예시 부탁: “게임 첫 화면을 초등학생도 이해하기 쉽게 바꾸고, 시작 버튼 아래에 조작 방법을 넣어 줘.”
2단계

GitHub 창고에 넣을 파일

  1. index.html: 지금 보고 있는 교재형 첫 화면입니다.
  2. game.html: 원래 파일을 바탕으로 넣은 한국사 전사 배틀 RPG입니다.
  3. README.md: GitHub에서 프로젝트 설명으로 보이는 안내문입니다.
GitHub Pages에서는 보통 index.html이 첫 화면으로 열립니다. 그래서 교재 화면을 첫 화면으로 두고, 게임은 버튼으로 들어가게 만들었습니다.
3단계

인터넷에 공개하기

  1. GitHub에 새 저장소를 만들고 이 폴더의 파일을 올립니다.
  2. 저장소의 Settings → Pages로 들어갑니다.
  3. Source를 GitHub Actions로 선택합니다.
  4. 잠시 기다리면 https://아이디.github.io/저장소이름/ 주소가 생깁니다.
